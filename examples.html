Working with plugins

Creating plugins

          aux --create_plugin=service myservice
          pip install -e aux_service_myservice/
        
          aux --create_plugin=device mydevice
          pip install -e aux_device_mydevice
        
          aux --create_plugin=protocol sip
          pip install -e aux_protocol_sip

Loading plugins

          from aux.system.service.ext import myservice
          from aux.system.device.ext import mydevice
          from aux.protocol.ext import sip
