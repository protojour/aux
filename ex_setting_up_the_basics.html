Setting up the basics

Installing AUX


Installing with pip from pypi

    https://pypi.python.org/pypi/aux
    pip aux
    -- or --
    sudo pip aux

Installing from git source

    git clone git@github.com:protojour/aux.git
    cd aux
    pip install -e .

The aux.properties file

    mkdir /home/$USER/.aux
    echo '{"logging":{"loggdirectory": "/home/user/.aux/"}}' > /home/$USER/.aux/aux.properties
