  • http
    • http.basic 
      http.basic( ('username', 'password') )                  
returns:
{'Authorization': 'Basic QWxhZGRpbjpvcGVuIHNlc2FtZQ=='}
usage:
headers.update(http.basic(credentials))
    • http.head 
      http.head(url="http://aux.com/api/resource", 
          headers={})
    • http.get 
      http.get(url="http://aux.com/api/resource", 
         headers={}, 
         body="", 
         request=None)
-- or -- 
You can use request=HTTPRequest instead of the other arguments,
works for all http.methods.
http.get(
request=HTTPRequest('http://aux.com/api/resource',
                    request_data={'method':'GET',
                                  'headers:{},
                                  'body'}
                    )
        )
